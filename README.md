# 🚀 CNN-Based Image Classification using PyTorch (CIFAR-10)

## 📌 Overview

This project implements a Convolutional Neural Network (CNN) using PyTorch to perform multi-class image classification on the CIFAR-10 dataset. The model is trained to classify images into 10 different categories such as airplane, car, bird, cat, dog, etc.

---

## 🧠 Key Features

* CNN model built from scratch
* Data preprocessing & normalization
* Training loop with forward & backward propagation
* Loss tracking per epoch
* Model evaluation (accuracy calculation)
* Clean and modular code structure

---

## 📂 Dataset

The project uses the CIFAR-10 dataset, which contains:

* 60,000 color images
* 10 classes
* Image size: 32×32

### Classes:

* Airplane
* Automobile
* Bird
* Cat
* Deer
* Dog
* Frog
* Horse
* Ship
* Truck
